WireGuard on Raspberry Pi 4B

A step-by-step guide to install and configure a secure WireGuard VPN

1. Update Your Raspberry Pi

First, update your Raspberry Pi to make sure you have the latest packages and kernel:

sudo apt update && sudo apt upgrade -y
sudo reboot

2. Install WireGuard

Install WireGuard using the official package manager:

sudo apt install wireguard -y

3. Enable IP Forwarding

Edit /etc/sysctl.conf and uncomment or add the following line:

sudo nano /etc/sysctl.conf
net.ipv4.ip_forward=1

Then apply the change:

sudo sysctl -p

4. Generate Keys

Create private and public keys for the WireGuard server:

wg genkey | tee privatekey | wg pubkey > publickey

5. Configure WireGuard Server

Create a config file at /etc/wireguard/wg0.conf:

[Interface]
PrivateKey = YOUR_PRIVATE_KEY
Address = 10.0.0.1/24
ListenPort = 51820
SaveConfig = true

[Peer]
# Example client
PublicKey = CLIENT_PUBLIC_KEY
AllowedIPs = 10.0.0.2/32

Replace YOUR_PRIVATE_KEY and CLIENT_PUBLIC_KEY accordingly.

6. Start and Enable WireGuard

sudo systemctl start wg-quick@wg0
sudo systemctl enable wg-quick@wg0

7. Firewall and NAT (Optional)

If you're using iptables to allow NAT, run:

sudo iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE
sudo iptables-save > /etc/iptables/rules.v4

8. Client Configuration

On your client (e.g., mobile or desktop), use the following config:

[Interface]
PrivateKey = CLIENT_PRIVATE_KEY
Address = 10.0.0.2/24

[Peer]
PublicKey = SERVER_PUBLIC_KEY
Endpoint = YOUR_PI_PUBLIC_IP:51820
AllowedIPs = 0.0.0.0/0
PersistentKeepalive = 25

9. Testing the Connection

Once everything is set up, you can verify the status:

sudo wg

Or to see interface info:

ip a show wg0
